# 🧠 Second Brain: Система синтеза знаний

Интеллектуальная база знаний (RAG), объединяющая нейробиологию (Сапольски), теорию риска (Талеб), когнитивистику (Каннеман) и психологию (Брене Браун).

## 🚀 Быстрый старт (Ежедневный цикл)

Чтобы обновить знания и задать вопрос, используй этот порядок:

1. **Пополнение**: Положи новые PDF в `data/import/`
2. **Синхронизация**: Запусти `update_brain.bat` (автоматически выполнит импорт, сбор статей, извлечение текста и индексацию).
3. **Анализ**: Задай вопрос через `ask.py`.

---

## 🛠 Справочник команд

### 📥 Сбор данных
* **Локальный импорт**:
    ```bash
    poetry run python -m brain2.cli.folder_import
    ```
    *Ищет PDF во всех подпапках `data/import`, считает хеши (защита от дублей) и регистрирует в базе.*
* **Сбор из сети**:
    ```bash
    poetry run bulk-collect
    ```
    *Автоматический поиск свежих статей в arXiv и Semantic Scholar.*

### ⚙️ Обработка
* **Извлечение текста**:
    ```bash
    poetry run brain2 extract
    ```
    *Превращает PDF в TXT. Пропускает уже обработанные файлы.*
* **Индексация (FAISS)**:
    ```bash
    poetry run brain2 index
    ```
    *Создает векторную память. На 20k+ фрагментов занимает около 2-5 минут.*
